import { useState } from 'react';
import { InlineCode } from './inline-code';

This component returns a highlighted text in the form of either a 'p', 'div', or 'span' element.

Using the Highlighted Text component:

```js live
<>
  This is a<InlineCode>element.</InlineCode>
  This is a<InlineCode>div element.</InlineCode>
  This is a<InlineCode>span element.</InlineCode>
</>
```

Choose one of seven available sizes from "xxs" to "xxl":

```js live
() => {
  const [size, setSize] = useState(16);

  return (
    <>
      <div>use font-size to change the component size.</div>
      <input type="range" min="10" max="40" value={size} onChange={(e) => setSize(e.target.value)} />
      <br />
      <br />
      <div style={{ fontSize: size + 'px' }}>
        <InlineCode>inline code</InlineCode>
      </div>
    </>
  );
};
```
