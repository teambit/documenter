### Overview

This component is built on `@teambit/documenter.theme.theme-context` and used for compositions inside @teambit projects.  
Using theme compositions to center element for example:

```js live
<ThemeCompositions>
  <div>This is a center element</div>
</ThemeCompositions>
```
